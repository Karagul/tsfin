Currency Future¶

Currency Future class to model currency futures. Also contains the BaseCurrencyFuture and CurrencyFutureHelper classes, which work as analogies to QuantLib’s instrument and helper classes, respectively. TODO: Propose implementation of these classes in QuantLib.

Try to deduce performance for a unit of the TimeSeries (as a financial instrument).

Warning

The result may be wrong. Use this method only for emergency/temporary cases. It is safer to instantiate a ‘real’ financial instrument class and call its performance method instead of using this.

Parameters:
  • start_date (datetime-like, optional) – The starting date of the period.
  • date (datetime-like, optional) – The ending date of the period.
  • start_quote (float, optional) – The quote of the instrument in start_date. Defaults to the quote in start_date.
